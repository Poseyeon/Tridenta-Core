# TridentaDB

## Build and Run

To build the project type this in your terminal

```bash
cargo build
```

To run the project type this in your terminal:

```bash
# CLI
cargo run --bin tridenta_db_cli

# Server
cargo run --bin tridenta_db_server

# Vite + React Frontend
npm run dev

# Electron App
npm start
```
